# Units

This section of the documentation shows a list of standard units. These units are used for all applicable Attributes within Odeon to simplify the conversion of these units and to enable a uniform calculation. Whenever an attribute is needed or given in a unit that differs from that standard set, the unit is appended in the attribute name.
For example, the attribute `internal_heat_capacity_j_per_k` contains the suffix `j_per_k` hinting that this attribute doesn't describe values in the standard unit *kWh/K*.

### List of standard units for Odeon

| Dimension              | Standard unit |
| ---------------------- | ------------- |
| temperature difference | K             |
| absolute temperature   | °C            |
| power                  | kW            |
| energy                 | kWh           |
| length                 | m             |
| area                   | m²            |
| volume                 | m³            |
| mass                   | kg            |
| time                   | h             |
| costs                  | EUR           |
| voltage                | V             |
| electric current       | A             |
| pressure               | Pa            |
| velocity               | m/s           |
| volume flow            | m³/s          |
| mass flow              | kg/s          |
